无法删除github上的文件的解决办法

发表于 2017-11-21 |
  1. 如果此文件夹已被加入git追踪，那么删除方法很简单，只需要将此文件夹删掉，然后提交一下就可以了

  2. 如果次文件夹曾经被加入过git追踪，现在被加入.gitignore里了，但是github上还有此文件夹。

    解决办法如下：

    git rm -r --cached 文件夹路径
git commit -m '删除忽略的目录 "文件夹名字"'
git push origin master

golangProtoGen问题

发表于 2017-11-03 |

这个问题的原因是：
在生成代码之前要先安装go get -u github.com/golang/protobuf/{proto,protoc-gen-go} 并且把GOPATH/bin目录下的protoc-gen-go放到环境变量PATH里面去，否则会报
protoc-gen-go: program not found or is not executable
这个错误。 命令如下：

$ go get -u github.com/golang/protobuf/{proto,protoc-gen-go}
